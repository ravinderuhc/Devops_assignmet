Task Management System

The Task Management System is a simple web application built using Flask, a lightweight web framework in Python. It allows users to manage their tasks by adding, viewing, updating, and deleting tasks through a web interface.

Features

Add Task: Users can add new tasks with a title and description.
View Tasks: Users can view all existing tasks.
Update Tasks: Users can update the title or description of existing tasks.
Delete Tasks: Users can delete tasks they no longer need.
